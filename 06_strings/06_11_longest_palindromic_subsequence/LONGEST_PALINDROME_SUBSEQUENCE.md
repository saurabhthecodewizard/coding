# Problem Name 
Longest Palindromic Sequence - [Leetcode - 5](https://leetcode.com/problems/longest-palindromic-substring/)

## Problem Description

Given a string s, return the longest palindromic substring in s.

## Difficulty - Medium

## Time Complexity
O(n^2)

## Space Complexity
O(1)

## Example 1:
```
Input: s = "babad"
Output: "bab"
Explanation: "aba" is also a valid answer.
```

## Example 2:
```
Input: s = "cbbd"
Output: "bb"
```
