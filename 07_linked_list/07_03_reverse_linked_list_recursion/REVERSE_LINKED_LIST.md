# Problem Name 
Reverse linked list - [Leetcode - 206](https://leetcode.com/problems/reverse-linked-list/)

## Problem Description

Given the head of a singly linked list, reverse the list, and return the reversed list.

## Difficulty - Easy

## Time Complexity
O(n)

## Space Complexity
O(1)

## Example 1:
```
Input: head = [1,2,3,4,5]
Output: [5,4,3,2,1]
```

## Example 2:
```
Input: head = [1,2]
Output: [2,1]
```
