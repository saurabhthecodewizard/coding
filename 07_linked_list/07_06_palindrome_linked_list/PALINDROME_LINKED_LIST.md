# Problem Name 
Palindrome Linked List - [Leetcode - 234](https://leetcode.com/problems/palindrome-linked-list/)

## Problem Description

Given the head of a singly linked list, return true if it is a
palindrome
or false otherwise

## Difficulty - Medium

## Time Complexity
O(2n)

## Space Complexity
O(1)

## Example 1:
```
Input: head = [1,2,2,1]
Output: true
```

## Example 2:
```
Input: head = [1,2]
Output: false
```
