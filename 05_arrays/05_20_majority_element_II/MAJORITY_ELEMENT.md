# Problem Name 
Majority Element II - [Leetcode - 229](https://leetcode.com/problems/majority-element-ii/)

## Problem Description

Given an integer array of size n, find all elements that appear more than ⌊ n/3 ⌋ times.

## Difficulty - Medium

## Time Complexity
O(n)

## Space Complexity
O(n)

## Example 1:
```
Input: nums = [3,2,3]
Output: [3]
```

## Example 2:
```
Input: nums = [1]
Output: [1]
```

## Example 3:
```
Input: nums = [1,2]
Output: [1,2]
```
