# Problem Name 
Pascal's Triangle - [Leetcode - 118](https://leetcode.com/problems/pascals-triangle/)

## Problem Description

Given an integer numRows, return the first numRows of Pascal's triangle.

## Difficulty - Medium

## Time Complexity
O(n)

## Space Complexity
O(n)

## Example 1:
```
Input: numRows = 5
Output: [[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]
```

## Example 2:
```
Input: numRows = 1
Output: [[1]]
```
